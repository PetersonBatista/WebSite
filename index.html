O Profissional Desenvolvedor

Um desenvolvedor de software é o profissional responsável por planejar, desenvolver e testar um software, aplicativo, website e até mesmo um sistema operacional (software que tem como papel gerenciar e administrar todos os recursos presentes em um sistema).

Esse profissional é um dos profissionais mais importantes em todo o setor de tecnologia, atuando juntamente com sua equipe, onde cada integrante fica responsável por desenvolver uma parte do projeto/sofware (front-end, design, back-end e a parte do banco de dados). Muito confuso ? Fique tranquilo, explicarei cada coceito logo a seguir!

Desenvolvedor Front-End

O profissional Front-End fica responsável por desenvolver o que está na "frente" de um site ou aplicativo, ou seja, o que fica visível para o usuário final. Essa parte do projeto é chamada de interface do usuário, desenvoldida para que as pessoas consigam navegar pela página ou aplicativo sem precisar ter conhecimentos avançados sobre a área da programação.

Para tornar uma plataforma fácil de usar, o desenvolvedor Front-End utiliza-se de Linguagens de Marcação de Hyper Texto, Linguagens de Programação, Software e várias Tecnologias. Como por exemplo:

HTML 5, É uma Linguagem de Marcação de Hipertexto utilizada para estruturar o conteúdo de um site (parte semântica) através de "tags", usadas para determinar o tipo de conteúdo que será exibido na página (texto, imagens, vídeos, etc.).

CSS3 (Cascading Style Sheets), Significa Folhas de Estilo em Cascata, basicamente o desenvolvedor Front-End utiliza dessas "Folhas de Estilo" para tornar o conteúdo apresentável ao usuário, alterando a cor da página, fonte das letras, tamanho de imagen/vídeos e o posicionamento dos conteúdos.

JavaScript, É uma Linguagem de Programação utilizada para dar "vida" a um site. Através desse Software o desenvolvedor consegue criar interatividade com os usuários, proporcionando uma navegação mais sociável.

Desenvolvedor Back-End

O profissional Back-End é o desenvolvedor que atua nos "bastidores", responsável por planejar, desenvolver e manter a estrutura de códigos que conecta a internet com o banco de dados (informações armazenadas), com também gerenciando as conexões dos usuários.

Python, É uma Linguagem de Progamação que atua em conjunto com o banco de dados e fluxos de informações, responsável pela criação de Softwares para análise e manipulação de informações.

Banco de Dados

Um banco de dados é um software responsável por colher e armazenar informações que sejam de interesse de um Gestor. Imagine uma escola com diversos alunos, onde cada um desses alunos possui informações relevantes para o ambiente acadêmico, como por exemplo: nome, matrícula, contato, notas, etc.

Esses dados ficam armazenados, onde através de Linguagens de Programação (Python) é possível fazer a manipulação desses dados e fornecer estatísticas para o gerenciamento e tomada de decisão pelos Professores, Secretários e Diretores desta escola.

MySQL é um sistema de gerenciamento de banco de dados, que utiliza a linguagem SQL como interface. Usado na criação e gerenciamento de bancos de dados baseados no modelo relacional.
